## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://maker.makecode.com/](https://maker.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/reemothman1/othman_reem_temp_project_de** and import

## Edit this project ![Build status badge](https://github.com/reemothman1/othman_reem_temp_project_de/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.
